## 重点服务事项

### 奖学金申请流程

奖学金申请通常包括通知发布、学生提交材料、学院初审、学校复核、公示和发放等环节。

**常见材料：**
- 学生本人基本信息与学籍信息
- 成绩单或绩点证明
- 获奖证明、科研竞赛证明或社会实践证明
- 班级或学院要求的补充材料

**办理建议：**
- 先确认奖学金类型和申请条件。
- 提前准备证明材料，避免公示前集中补交。
- 如不确定是否符合条件，可先进行材料预审。

### 图书馆研讨间预约

图书馆研讨间面向学习小组、课程讨论、项目答辩准备等场景开放。

**预约规则：**
- 需要提供预约人 userId、预约日期、时间段和人数。
- 每次预约默认占用 1 个名额。
- 如遇考试周或资源紧张，建议提前 1-3 天预约。
- 使用结束后应保持室内整洁。

**可办理方式：**
- 线上预约
- 线下服务台确认
- 自助终端预约

### 心理咨询预约

心理咨询服务用于学习压力、人际关系、情绪困扰、生涯规划等问题的初步支持。

**预约说明：**
- 咨询内容会按学校规则进行隐私保护。
- 如用户表达强烈危机信号，应优先建议联系学校心理中心或紧急联系人。
- 智能助手只负责预约和流程说明，不替代专业心理咨询师判断。

### 在读证明办理

在读证明用于实习、签证、竞赛报名、资格审核等场景。

**办理流程：**
1. 确认学生身份和学籍状态。
2. 提交用途说明。
3. 选择线上下载或线下盖章领取。
4. 等待教务部门审核。

**注意事项：**
- 如需要盖章原件，应选择线下窗口领取。
- 如仅用于线上报名，可优先选择电子版证明。

### 校园卡补办

校园卡补办适用于丢失、损坏、磁条异常、照片信息错误等情况。

**办理说明：**
- 补办前建议先挂失，避免余额被误用。
- 补办可能产生工本费。
- 领取方式通常包括自助机领取或校园卡中心窗口领取。

### 体育馆预约

体育馆预约用于篮球场、羽毛球场、乒乓球室等场地资源。

**预约规则：**
- 需要选择场地类型、时间段和人数。
- 高峰时段可能限制单人连续预约次数。
- 如无法按时使用，应及时取消预约。

### 宿舍报修

宿舍报修适用于水电、门锁、空调、网络、家具等宿舍设施问题。

**报修信息建议包含：**
- 宿舍楼、房间号
- 问题类型和故障描述
- 是否影响安全或基本生活
- 方便维修人员联系的时间段

## 回答规范

当用户咨询政策或流程时，应优先说明适用场景、办理步骤、所需材料、注意事项。若用户想直接预约或办理，应交给事务办理 Agent 处理；若用户表达不满、投诉或建议，应交给反馈 Agent 处理。
